# Desafio_Dio_Transcritor-de-voz
Esse código em Python grava cinco segundos de áudio pelo microfone, salva o resultado em um arquivo chamado entrada.wav, envia esse arquivo para o modelo Whisper da OpenAI para realizar a transcrição automática da fala em texto e exibe no console o conteúdo transcrito, permitindo visualizar em palavras o que foi dito durante a gravação.
